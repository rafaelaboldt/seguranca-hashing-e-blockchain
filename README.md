# Demonstrações de Segurança e Blockchain

Curso de Análise e Desenvolvimento de Sistemas - UniSenac
Disciplina: Algoritmos e Estrutura de Dados II (4º semestre)

Este repositório contém três scripts de exemplo usados em aula para demonstrar conceitos básicos de segurança e hashing:

- `senhas.py`: demonstração de geração e verificação de senha usando PBKDF2 com salt.
- `git.py`: demonstração de geração de hashes (SHA-1) para versões de conteúdo, simulando como o Git identifica mudanças.
- `blockchain.py`: implementação simples de uma blockchain com três blocos, demonstra a dependência entre hashes e como alterações quebram a cadeia.

Como executar

Abra um terminal e execute:

```bash
python senhas.py
python git.py
python blockchain.py
```
